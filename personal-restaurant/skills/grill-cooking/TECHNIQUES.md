기본 굽기 순서:
1. 고기 상온 적응 (30분)
2. 물기 제거 + 시즈닝
3. 그릴/팬 예열 (고온)
4. 기름칠 (고기에, 팬 아님)
5. 첫 면 굽기 (만지지 않기)
6. 뒤집기 (한 번만)
7. 두 번째 면 굽기
8. 휴지 (조리시간의 절반)

뒤집기 타이밍:
- 자연 분리: 고기가 팬에서 쉽게 떨어짐
- 갈색 크러스트: 표면에 마이야르 반응
- 옆면 색 변화: 1/3 정도 익음 표시

휴지 (Resting):
- 육즙 재분배: 절단 시 육즙 손실 방지
- 내부 온도 상승: 잔열로 2-3°C 추가 상승
- 근육 이완: 더 부드러운 식감

완성 기준:
| 감각 | 기준 |
|------|------|
| 시각 | 균일한 갈색 크러스트 |
| 촉각 | 목표 익힘의 탄력 |
| 청각 | 자글거림 유지 |
| 후각 | 마이야르 향 |
